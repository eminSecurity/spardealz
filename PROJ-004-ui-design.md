# PROJ-004: UI/UX Design System

## Design-Richtlinien

### Farben
- **Primary:** #FF6B35 (Orange - wie MyDealz, aber moderner)
- **Secondary:** #1A1A2E (Dunkelblau/Anthrazit)
- **Success:** #22C55E (Grün für Hot Deals)
- **Danger:** #EF4444 (Rot für abgelaufen)
- **Background:** #F8FAFC (Hellgrau)
- **Surface:** #FFFFFF (Weiß)

### Layout
- **Max-Width:** 1400px
- **Card-Grid:** Responsive (1-4 Spalten)
- **Spacing:** 4px-Grid-System
- **Border-Radius:** 12px (Cards), 8px (Buttons)

### Typografie
- **Headlines:** Inter, 600-700 weight
- **Body:** Inter, 400 weight
- **Code/Mono:** JetBrains Mono

## Komponenten

### DealCard
- Thumbnail (16:9)
- Preis-Badge (original + reduziert)
- Temperatur-Indicator (Hot/Kalt)
- Merchant-Name
- Zeitstempel
- Vote-Buttons

### FilterBar
- Kategorie-Dropdown
- Preis-Range Slider
- Sortierung (Neueste, Heißeste, Preis)
- Suche

### DealDetail
- Hero-Bild (groß)
- Preisvergleich-Widget
- Deal-Info (Händler, Gültigkeit)
- Kommentar-Section
- Ähnliche Deals

### Navigation
- Sticky Header
- Logo links
- Suche mittig
- User-Menu rechts
- Kategorie-Links darunter

## Animations
- Card-Hover: Scale 1.02 + Shadow
- Button-Hover: Background-Shift
- Page-Transitions: Fade 200ms
- Loading: Skeleton-Shimmer
